Cursor launched a public beta for iOS, letting developers control cloud and local agents from anywhere. The developer-tool race is shifting from code completion toward remote delegation.

Cursorは2026年6月29日にiOS向け公開ベータを開始し、Paid planのユーザーは外出先から常時稼働エージェントを起動・管理できるようになった。[1] モバイル版ではリポジトリを選んでクラウドエージェントを立ち上げ、音声入力やスラッシュコマンドで指示しながら、開発環境付きの仮想マシン上でテストや検証まで回せる。[2] さらに、ローカルPC上のエージェントをスマホから継続操作するRemote Controlも用意されており、ラップトップを開けない状況でも作業を止めにくい設計になっている。[2][3] この展開は、CursorがWeb・Slack・GitHub・Jiraへと接点を広げてきた流れの延長であり、開発者は“どの画面からでも同じエージェントに指示する”体験へ近づいている。[3] 一方で、AnthropicのClaude Codeはコードベース読解から変更・テスト・コミットまでを担うエージェント型で、GitHub CopilotもSDKやアプリ展開を強めており、競争は補完機能よりも「どこまで自律実行できるか」に移っている。[4]
