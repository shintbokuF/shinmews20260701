AzureでClaudeがFoundryに正式提供、GB300 Blackwell Ultra上でホスト

3行でわかる！今日のニュース

  • ClaudeがFoundryで一般提供に
  • Azure上でGB300により稼働
  • 既存Azure管理で導入を容易化

ClaudeがMicrosoft Foundryで一般提供となり、Azure上でGB300 Blackwell Ultraによりホストされる体制が整った。企業は既存のAzure基盤のままClaudeを使い、認証・課金・ガバナンスを統合できる。

Claude is now generally available in Microsoft Foundry, hosted on Azure and running on NVIDIA GB300 Blackwell Ultra. Enterprises can use Claude within their existing Azure stack with integrated auth, billing, networking, and governance.

Microsoft Azure Blogは、AnthropicのClaudeがMicrosoft Foundryで一般提供になり、Azure上でホストされ、NVIDIA GB300 Blackwell Ultra上で稼働すると伝えた[1][2]。Microsoftは、Azureアカウントに紐づく認証・課金・ネットワーク・ガバナンスをそのまま使えるため、企業はインフラ調達よりもエージェント開発に集中できると説明している[2][3]。Anthropic側も、MicrosoftとNVIDIAとの戦略提携を通じてAzure上でClaudeの提供範囲を広げ、Azure企業顧客のモデル選択肢と新機能を拡張すると表明した[4]。この流れは、FoundryがAzure OpenAI系とAnthropic系の両方を扱う統合AI基盤として存在感を強めることを示している[3]。
